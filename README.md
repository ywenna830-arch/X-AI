## 环境要求

- Windows 10 或 Windows 11
- Python 3.10 或更高版本

## 本地运行

1. 创建虚拟环境：

```powershell
python -m venv .venv
```

2. 激活虚拟环境：

```powershell
.\.venv\Scripts\Activate.ps1
```

3. 安装依赖：

```powershell
python -m pip install -r requirements.txt
```

4. 准备本地环境变量文件：

```powershell
copy .env.example .env
```

如需接入真实AI接口，请在 `.env` 中配置：

```text
AI_API_KEY=你的本地密钥
AI_API_BASE_URL=OpenAI兼容的chat/completions地址
AI_MODEL=模型名称
AI_TIMEOUT=20
MAX_IMPORT_FILE_BYTES=5242880
APP_TIMEZONE=Asia/Shanghai
```

未配置 `AI_API_KEY` 时，文字通知解析会启用页面明确标注的本地演示解析模式，项目不会崩溃。

图片OCR依赖 `pytesseract` Python包和本机 Tesseract OCR 程序；未安装时应用仍可启动，上传图片时会显示明确提示。

5. 启动项目：

```powershell
python run.py
```

如果本机 5000 端口已被占用，可以临时指定其他端口：

```powershell
$env:PORT=5050
python run.py
```

6. 访问地址：

- 首页：http://127.0.0.1:5000/
- 添加任务：http://127.0.0.1:5000/tasks/new
- AI识别确认：http://127.0.0.1:5000/tasks/confirm
- 任务中心：http://127.0.0.1:5000/tasks
- 智能计划：http://127.0.0.1:5000/plan
- 健康检查：http://127.0.0.1:5000/health

## 运行测试

```powershell
python -m pytest
```

## 当前功能

- Flask 应用可启动
- 首页、添加任务、AI识别确认、任务中心和智能计划页面可切换
- SQLite 持久化保存任务，数据库位于本地 `instance/` 目录
- 支持手动新增、查看、编辑、删除、筛选和状态切换任务
- 首页读取真实任务和计划数据，展示今日任务、明日任务、即将截止、应用内提醒、已逾期和容量不足风险
- 添加任务页已改为对话式主入口，并保留真实可用的手动填写入口
- 文字通知AI解析已接入阶段2.5的对话式输入框，解析后进入确认页，用户确认后才保存
- 未配置API密钥时提供明确标注的本地演示解析模式
- 智能计划页支持保存未来7天或14天每日可用时间、提前完成天数、周末加时和不可安排日期
- 规则规划器可按任务截止时间、剩余时长、优先级、当前状态和每日80%容量生成待确认计划
- 支持课程论文、PPT展示、编程作业、在线课程、日常练习和考试复习的基础拆解模板
- 用户确认后计划项保存到SQLite，计划页展示真实已保存周计划和容量不足预警
- 智能计划页支持记录计划项执行反馈：未开始、已完成、部分完成和未完成
- 部分完成可填写已完成分钟数或比例，未完成可选择原因并触发剩余计划重排
- 动态重规划复用阶段4规则算法，只调整未开始、部分完成和未完成的剩余计划
- 已完成计划项的日期、分钟数、状态和历史记录会保留，不会被修改、删除或重新安排
- 延后一天和修改预计时长会检查截止日期、每日容量并记录调整原因
- 添加任务页的“+”附件菜单已接入图片、DOCX和PDF文字提取
- 图片支持PNG、JPG、JPEG，通过OCR提取文字后先展示原文，用户修改后再进入现有AI解析流程
- Word仅支持DOCX，提取正文段落和表格文字；不支持旧版DOC
- PDF使用PyMuPDF提取文本型PDF文字并记录页码来源；扫描型PDF会提示当前不做整份PDF OCR
- 上传文件默认限制为5MB，可通过 `MAX_IMPORT_FILE_BYTES` 调整；文件仅临时保存，解析成功或失败后都会清理
- 上传会校验扩展名和基础文件签名，并使用安全文件名，避免路径穿越
- 每个任务可设置提前7天、3天、1天、截止当天或自定义0-365天的应用内提醒
- 应用内提醒需要用户打开网页才能查看；第一版不是短信、微信、邮件或原生手机后台推送
- 浏览器通知只能由用户主动点击启用，依赖浏览器支持和权限；网页关闭后不保证通知
- 已完成任务不会继续产生新的应用内提醒，也不会默认进入ICS导出
- 支持下载单个任务ICS和全部未完成任务ICS；任务详情页可下载单任务，首页可下载全部未完成任务
- ICS 默认时区为 `Asia/Shanghai`，可通过 `APP_TIMEZONE` 调整；只有日期没有具体时间时默认23:59为截止时间
- 已有计划项的ICS事件默认09:00开始，按计划项分钟数计算结束时间；无计划项任务使用截止时间生成30分钟事件
- ICS 会根据任务提醒设置生成 `VALARM`，并转义中文文本中的逗号、分号、反斜杠和换行
- `/health` 返回健康检查状态

## ICS导入说明

- iPhone日历：下载 `.ics` 文件后用系统日历打开，按提示添加到日历。
- Android或Google日历：在 Google Calendar 网页端选择“设置”→“导入和导出”→导入 `.ics` 文件。
- Outlook：选择“添加日历”或“导入/导出”，导入下载的 `.ics` 文件。
